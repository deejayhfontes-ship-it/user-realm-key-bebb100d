# NUPPE — Base de Conteúdo para Gerador de Peças Social Media

## Objetivo
Este arquivo serve como base de conteúdo para a criação de uma **nova aba** dentro da página do NUPPE chamada **Gerador de Peças Social Media**.

O módulo deve transformar o conteúdo institucional, campanhas e cursos em peças editáveis para Instagram, sempre com exportação nos formatos:
- **1080x1440**
- **1080x1920**
- **1080x1080**

## Regras obrigatórias
- Criar esta funcionalidade em uma **nova aba** da página do NUPPE.
- Usar **obrigatoriamente as logos anexadas** ao projeto.
- Permitir **upload de imagens** para uso como **fundos dos criativos**, especialmente nas campanhas.
- A linguagem visual não pode ser clichê, genérica ou escolar padrão.
- O resultado deve parecer **moderno, estratégico, atual, institucional e comercial**, conforme o contexto.

---

## Estrutura esperada do módulo

### Abas internas
- Campanhas
- Cursos
- Biblioteca de peças
- Exportação

### Entregas principais
1. **3 campanhas completas**
2. **1 peça individual para cada curso**
3. **Variações automáticas** da mesma peça nos 3 formatos
4. **Edição manual** sem quebrar o layout
5. **Exportação em PNG** de alta qualidade
